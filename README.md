# Complex Modifications

## 光标定位

直接移动到行首\行尾	->	Control + i\e
方向键移动	->	Control + njkl
新起一行	->	Control + enter

## 输入法切换

单击Shift 切换输入法	->	left_shift

## LOOP

激活Loop 应用	->	Control + g
